# Video Generation Tool (Remotion + Gemini + Veo)

**Status:** Complete
**Mode:** Thorough
**Created:** 2026-03-26

## Summary

Build a **set of video production primitives** that an AI agent can orchestrate to create videos of any length — from 8-second clips to 2-hour films.

The primitives are:
- **`generate_asset`** — Generate still images from prompts (Gemini 3.1 Flash Image via OpenRouter)
- **`generate_clip`** — Generate short video clips from text OR from a reference image (Veo via Google AI Studio)
- **`stitch_clips`** — Concatenate clips into a final video (ffmpeg — simple, fast)
- **`compose_video`** — Compose clips with overlays, transitions, text, music (Remotion — rich, flexible)
- **`scaffold_project`** — Create a Remotion project for complex compositions

The **agent** is the director:
- During **planning**, it writes the story, scene breakdown, visual style guide, and character descriptions
- During **building**, it loops scene-by-scene: generate reference image → image-to-video → collect clip
- The skill file teaches the agent this workflow

This enables consistent, long-form video production through the standard plan/build agent loop.

## Key Architectural Principle

```
┌──────────────────────────────────────────────────────────────┐
│                    AGENT (Plan/Build Loop)                     │
│                                                                │
│  Plan Phase:                                                   │
│    - Story outline, scene descriptions                         │
│    - Visual style guide (for image consistency)                │
│    - Character/setting descriptions (reused across scenes)     │
│    - Scene-by-scene breakdown with prompts                     │
│                                                                │
│  Build Phase (per scene):                                      │
│    1. generate_asset("Scene 1: wide shot of...")  → scene1.png │
│    2. generate_clip(prompt, ref=scene1.png)        → scene1.mp4│
│    3. ... repeat for all scenes ...                            │
│    4. stitch_clips([scene1.mp4, scene2.mp4, ...])  → final.mp4│
│       OR                                                       │
│    4. compose_video(script_with_transitions)        → final.mp4│
│                                                                │
│  The agent maintains cohesion through:                         │
│    - Consistent style prefixes in all prompts                  │
│    - Reference images as first frames for video clips          │
│    - Scene-to-scene continuity in prompt descriptions          │
└──────────────────────────────────────────────────────────────┘
```

**Simple path** (no Remotion): image gen → image-to-video → ffmpeg concat
**Rich path** (Remotion): image gen → image-to-video → Remotion compose with overlays/transitions/text/music

## Requirements

1. Generate still image assets from text prompts using Gemini 3.1 Flash Image via OpenRouter API
2. Generate video clips from text prompts OR from reference images (image-to-video) using Google Veo
3. Concatenate video clips into a final video using ffmpeg (simple path)
4. Optionally compose video with overlays, transitions, text using Remotion (rich path)
5. Support multiple aspect ratios (16:9, 9:16) and any duration via clip chaining
6. Work as OpenCode custom tools callable by the LLM
7. Also work as a Randal tool + skill in the Randal repo
8. The skill file teaches the agent the full production workflow

## Constraints

- Image generation: OpenRouter API — model `google/gemini-3.1-flash-image-preview`
- Video generation: Google AI Studio Veo API — async submit+poll pattern, 4-8s clips
- Veo clips are max 8 seconds — longer videos are built by chaining clips
- ffmpeg must be available on PATH for the stitch tool
- Remotion is optional — only needed for `compose_video`
- Environment variables: `OPENROUTER_API_KEY`, `GOOGLE_AI_STUDIO_KEY`
- Output directory: configurable via `VIDEO_OUTPUT_DIR` (default: `/tmp/video-gen/`)

## API Reference (for implementation)

### OpenRouter — Gemini 3.1 Flash Image
```
POST https://openrouter.ai/api/v1/chat/completions
Authorization: Bearer $OPENROUTER_API_KEY
Content-Type: application/json

{
  "model": "google/gemini-3.1-flash-image-preview",
  "messages": [
    { "role": "user", "content": "Generate an image of: {prompt}" }
  ]
}
```
Response includes base64-encoded image in the message content.

### Google AI Studio — Veo (Text to Video)
```
POST https://generativelanguage.googleapis.com/v1beta/models/veo-3.0-generate-001:predictLongRunning
x-goog-api-key: $GOOGLE_AI_STUDIO_KEY
Content-Type: application/json

{
  "instances": [{ "prompt": "{prompt}" }],
  "parameters": {
    "aspectRatio": "16:9",
    "durationSeconds": 8,
    "resolution": "720p",
    "sampleCount": 1
  }
}
```

### Google AI Studio — Veo (Image to Video)
```
POST https://generativelanguage.googleapis.com/v1beta/models/veo-3.0-generate-001:predictLongRunning
x-goog-api-key: $GOOGLE_AI_STUDIO_KEY
Content-Type: application/json

{
  "instances": [{
    "prompt": "{prompt}",
    "image": { "bytesBase64Encoded": "{base64_image}" }
  }],
  "parameters": {
    "aspectRatio": "16:9",
    "durationSeconds": 8,
    "resolution": "720p",
    "sampleCount": 1
  }
}
```
Returns operation name → poll until complete → download video.

### ffmpeg — Concat Clips
```bash
# Simple concat (no transitions)
ffmpeg -f concat -safe 0 -i filelist.txt -c copy output.mp4

# With crossfade transitions (2 clips)
ffmpeg -i clip1.mp4 -i clip2.mp4 -filter_complex "xfade=transition=fade:duration=1:offset=7" output.mp4
```

### Remotion SSR Rendering
```typescript
import { bundle } from '@remotion/bundler'
import { renderMedia, selectComposition } from '@remotion/renderer'

const bundled = await bundle({ entryPoint: './src/index.ts' })
const comp = await selectComposition({ serveUrl: bundled, id: 'ScriptedVideo', inputProps })
await renderMedia({ composition: comp, serveUrl: bundled, codec: 'h264', outputLocation: 'out.mp4', inputProps })
```

## Video Script Schema (for the ScriptedVideo Remotion composition)

```typescript
interface VideoScript {
  title?: string
  fps?: number            // default: 30
  width?: number          // default: 1920
  height?: number         // default: 1080
  scenes: Scene[]
}

interface Scene {
  type: "image" | "video" | "text" | "color"
  src?: string            // path to image or video file (relative to public/)
  text?: string           // text content for "text" type
  color?: string          // background color for "color" type
  duration: number        // duration in seconds
  transition?: {
    type: "crossfade" | "slide-left" | "slide-right" | "zoom-in" | "cut"
    duration: number      // transition duration in seconds
  }
  overlay?: {
    text?: string
    position?: "top" | "center" | "bottom"
    style?: "title" | "caption" | "subtitle"
  }
}
```
